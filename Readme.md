### How use/testing

Install dependencies IMS-Service

```console
~/Projects/IMS$ cd IMS-Service
~/Projects/IMS/IMS-Service$ npm install
~/Projects/IMS/IMS-Service$ cd ..
~/Projects/IMS$ docker-compose up
```
